Detailed Description

A service for managing and rendering the user interface (UI) in the game.

The CUIService class handles all aspects of the user interface, including initialization, rendering, and interaction with the user. It integrates with DirectX 11 and ImGui to provide a graphical interface for controlling various features of the game.
