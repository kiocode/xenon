Detailed Description

Configuration for the ESP (Extra Sensory Perception) feature, controlling the visual rendering of enemy information.

This class defines the settings for rendering visual elements such as snaplines, 2D and 3D boxes, health bars, and skeletons. It also includes color settings for each visual element and custom functionality for obtaining bone screen positions.
