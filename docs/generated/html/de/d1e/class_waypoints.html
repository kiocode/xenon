Detailed Description

A class that manages and renders waypoints in both world space and radar view.

The Waypoints class extends from CComponent and provides functionality for storing, rendering, and interacting with waypoints in a game or application. It supports both world and radar views, allowing waypoints to be set in 2D or 3D space.
